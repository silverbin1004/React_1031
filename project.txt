### **React의 핵심 개념 이해**

- **질문**: React에서 컴포넌트란 무엇이며, 어떤 두 가지 형태로 작성할 수 있나요?
- **답변**: 컴포넌트란 독립적이고 재사용 가능한 단위를 의미한다. 컴포넌트를 정의하는 방법에는 함수형 컴포넌트와 클래스형 컴포넌트가 있다.

### **JSX의 역할**

- **질문**: JSX란 무엇이며, React에서 어떤 역할을 하나요?
- **답변**: JavaScript를 확장한 문법으로 HTML과 유사한 구조를 자바스크립트 코드 내에서 사용할 수 있도록 해준다. React에서는 UI를 정의하는 데 사용된다.

### **State와 Props의 차이**

- **질문**: React에서 **state**와 **props**의 차이점을 설명하세요.
- **답변**: props는 컴포넌트가 외부에서 받는 데이터를 의미하고, 읽기 전용이기 때문에 컴포넌트 내부에서 직접 수정을 할 수 없다.
           state는 컴포넌트 내부에서 관리되는 데이터를 의미하며, 읽기와 쓰기 모두 가능하다.

### **이벤트 처리 방식**

- **질문**: React에서 이벤트를 처리할 때 주의해야 할 두 가지 사항은 무엇인가요?
- **답변**: 이벤트 이름은 카멜케이스로 작성해야 하며 이벤트 핸들러는 함수로 전달해야 한다.

### **컴포넌트 내 조건부 렌더링**

- **질문**: 컴포넌트에서 조건부 렌더링을 구현하는 일반적인 방법 두 가지는 무엇인가요?
- **답변**: 1. if문을 사용한다. 2. 삼항 연산자를 사용한다.

### **리스트 렌더링과 Key**

- **질문**: React에서 배열을 렌더링할 때 각 요소에 `key`를 부여하는 이유는 무엇인가요?
- **답변**: 각 요소를 고유하게 식별하여 성능을 최적화 시키기 위해서이다.

### **useState Hook의 기본 사용법**

- **질문**: 함수형 컴포넌트에서 `useState` Hook을 사용하여 상태를 관리하는 기본적인 방법을 설명하세요.
- **답변**: useState Hook을 호출하여 상태 변수와 상태를 갱신하는 함수를 반환받는다.

### **useEffect Hook의 역할**

- **질문**: `useEffect` Hook은 어떤 용도로 사용되나요?
- **답변**: 함수형 컴포넌트에서 부수 효과를 수행하기 위해 사용된다.

### **Props로 함수 전달하기**

- **질문**: 부모 컴포넌트에서 자식 컴포넌트로 함수를 props로 전달하는 이유는 무엇인가요?
- **답변**: 부모 컴포넌트가 자식 컴포넌트의 이벤트나 상태 변경을 제어하고 응답할 수 있다.

### **컴포넌트의 생명주기 메서드**

- **질문**: 클래스형 컴포넌트에서 컴포넌트가 마운트된 직후에 호출되는 생명주기 메서드는 무엇인가요?
- **답변**: componentDidMount() 가 호출된다.

### **고차 컴포넌트(HOC)의 기본 개념**

- **질문**: 고차 컴포넌트(HOC)란 무엇인가요?
- **답변**: 고차 컴포넌트는 컴포넌트 로직을 재사용하기 위한 React의 고급 기술로 컴포넌트를 가져와 새 컴포넌트를 반환한다.

### **고차 컴포넌트의 네이밍 컨벤션**

- **질문**: HOC의 이름을 지을 때 일반적으로 어떤 접두사를 사용하나요?
- **답변**: 'with'

### **Context API의 목적**

- **질문**: React에서 Context API는 어떤 문제를 해결하기 위해 사용되나요?
- **답변**: props drilling 문제를 해결하기 위해 사용된다.

### **Redux의 주요 요소**

- **질문**: Redux에서 상태 관리를 위한 세 가지 핵심 요소는 무엇인가요?
- **답변**: store, actions, reducers 
           store - 애플리케이션의 전체 상태를 하나의 객체 트리로 저장하는 중앙 저장소.
           actions - 애플리케이션 데이터에 접근하거나 데이터를 조작하는 실행 단위.
           reducers - 스토어에 저장되어 있는 상태를 실제로 조작하는 함수.

### **React Router의 기본 사용법**

- **질문**: React Router에서 URL 경로에 따라 컴포넌트를 렌더링하기 위해 사용하는 컴포넌트는 무엇인가요?
- **답변**: Route 컴포넌트 -> 특정 url 경로와 해당 경로에 대응하는 컴포넌트를 연결한다.
